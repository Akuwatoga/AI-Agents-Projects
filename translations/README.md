# Translation workspace

English source files remain in their original locations. Localized content mirrors those paths
under a locale directory, for example:

```text
translations/zh-CN/README.md
translations/zh-CN/agents/01-web-research-agent/README.md
translations/zh-CN/agents/01-web-research-agent/metadata.yaml
translations/zh-CN/crewai_mcp_course/README.md
```

Do not translate Python, shell commands, URLs, file names, environment variables, package names,
framework names, or model identifiers. See `docs/LOCALIZATION_ZH_CN.md` for the complete workflow.
